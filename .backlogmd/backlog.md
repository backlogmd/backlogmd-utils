# Roadmap

## Features

### 001 - BacklogMD parser emits JSON

- **Status:** done
- **Feature:** [BacklogMD parser emits JSON](features/backlogmd-parser-emits-json/index.md)
- **Description:** Convert BacklogMD markdown (backlog, features, tasks) into a single canonical JSON file with cross-links and validation.

### 002 - Remote Markdown File Support

- **Status:** todo
- **Feature:** [Remote Markdown File Support](features/backlogmd-parser-emits-json/index.md)
- **Description:** CLI should support fetching and parsing remote markdown files via HTTP/HTTPS URLs

### 003 - Dev Board Server

- **Status:** done
- **Feature:** [Dev Board Server](features/dev-board-server/index.md)
- **Description:** Lightweight dev server that watches `.backlogmd/` and serves a live-updating kanban board with Todo, In Progress, and Done columns.
